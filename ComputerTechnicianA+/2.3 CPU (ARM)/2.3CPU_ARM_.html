New Terminology

Single Board Computer (SBC)

A computer where almost everything is on one single board.

Raspberry Pi

Want one? Pi 5, 8GB

The most popular and stable SBC today.

 

Jetson Nano

Nvidia’s version of a SBC. It is like the Pi with a GPU.

CPU Architectures

Sets of instructions (Instruction Set Architecture, also known as an ISA) that processors use to perform tasks and communicate with hardware, each having its own unique architecture and design principles.

x86

RISC-V

A proprietary ISA that Intel and AMD use.

  • x86_64 is 64 bit

Part I - Imaging the microSD card for the Pi

Open the Raspberry Pi Imager

If you don’t have it, you can download it here -  https://www.raspberrypi.com/software/

Press "CTRL+Shift+X" to access a hidden menu. Select the following:

  • Set Hostname to FirstName.LastName_Pi
  • Set Username and Password to whatever you want
  • Configure WiFi to your Home's WiFi & Password
  • WiFi Country - US
  • Set Locale Settings to America/Denver
  • Skip first-run Wizard

Choose an OS (note, Raspberry Pi OS Lite is terminal only) - Raspberry Pi OS (32-bit) - Debian Bullseye is recommended.

Flash the SD/microSD card with the image you wanted. When finished, put the SD/microSD card into your Pi.

When it finishes, put the microSD card into the Raspberry Pi and power it on.

Connect the following:

  • HDMI micro to HDMI cable to your capture card.
  • Ethernet cord to the Cisco switches on your table.
  • Keyboard and mouse.
